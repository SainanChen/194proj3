3.1 Exaggerate Color

To exaggerate color features on a face, we can always get the features first by subtracting the warpped face from the main face, and then add or subtract the difference times a scale (alpha) back to the main face.

It seems not work not on the gray images. Now I will try to exaggerate shape features instead.

3.2 Exaggerate Shape

To exaggerate shape features on a face, I times a scale (alpha) to difference between the reference points on my face and the mean face. Then I add the result back to mean face reference point cooridinates to enlarge my difference from the mean face. 

Below are examples setting alpha = 1.5 and 3 respectively, and we can see my features like  large nose, smiled mouth and narrower jaw are exaggerated!
